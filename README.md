Input Program dan Penilaian Nilai Mahasiswa
Program ini dirancang untuk memasukkan data siswa ke dalam sebuah daftar secara berulang dan menghitung Nilai Akhir berdasarkan bobot yang telah ditentukan

#Meminta input data secara berulang (menggunakan perulangan).

Menambahkan data yang diinput ke dalam struktur data (list).

Menampilkan pertanyaan untuk menambah data (y/t? ) sebagai kontrol perulangan.

Menghitung nilai akhir dengan bobot yang spesifik.

#Alur Program dan Input

Program akan berjalan dengan alur

Inisialisasi Daftar: Program membuat daftar kosong untuk menyimpan data setiap siswa.

Peruangan Input: Program masuk ke dalam perulangan tak terbatas (atau do-while):

Meminta input untuk data mahasiswa (Nama, NIM, Nilai Tugas, Nilai UTS, Nilai UAS).

Melakukan perhitungan Nilai Akhir .

menyimpan data lengkap tersebut ke dalam daftar.

Menampilkan pertanyaan: "Tambah data (y/t)?"

Keputusan :

Jika menjawab y(Ya), perulangan berlanjut (kembali ke langkah 2).

Jika menjawab t(Tidak), perulangan  berhenti.

Output: Program menampilkan seluruh daftar data siswa yang telah dikumpulkan dalam daftar.

#Perhitungan Nilai Akhir

Nilai Akhir dihitung dengan 
tiga nilai komponen dengan pembobotan yang sudah tetap:

Nilai Tugas	30%
Nilai UTS	35%
Nilai UAS	35%
Formula Perhitungan:

Nilai Akhir=( 0,30Nilai Tugas )+( 0,35Nilai UTS )+( 0,35Nilai UAS )
